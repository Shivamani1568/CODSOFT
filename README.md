# CODSOFT

My submissions for the CodSoft Machine Learning Internship.

Three tasks in here, each in its own folder. Each folder has its own README with the details and how to run it.

## Tasks

1. **codsoft_task1_credit_card_fraud_detection** — credit card fraud detection using Logistic Regression, Decision Tree, and Random Forest. 
2. **codsoft_task2_customer_churn_prediction** — customer churn prediction for a bank using Logistic Regression, Random Forest, and Gradient Boosting. 
3. **codsoft_task3_spam_sms_detection** — SMS spam classification using TF-IDF features with Naive Bayes, Logistic Regression, and Linear SVM on the UCI SMS Spam Collection dataset. 

## Setup

Each task uses Python 3 with pandas, numpy, scikit-learn, and matplotlib. Setup steps are in each task's README.
